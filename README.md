#### AI/ML Security

[**Sovereign-RAG-Gateway**](https://github.com/ogulcanaydogan/Sovereign-RAG-Gateway) - OpenAI-compatible gateway that enforces policies, redacts PHI/PII, and controls retrieval access for regulated AI workloads.

[**LLM-Supply-Chain-Attestation**](https://github.com/ogulcanaydogan/LLM-Supply-Chain-Attestation) - Sigstore-based attestation framework for LLM pipelines. Signs and verifies prompts, training data, evaluations, and model artifacts.

[**Prompt-Injection-Firewall**](https://github.com/ogulcanaydogan/Prompt-Injection-Firewall) - Detection middleware with 50+ patterns mapped to OWASP LLM Top 10.

[**Verifiable-AI-Output-Ledger**](https://github.com/ogulcanaydogan/Verifiable-AI-Output-Ledger) - Append-only audit ledger for AI outputs using DSSE envelopes and Merkle proofs.

[**AI-Provenance-Tracker**](https://github.com/ogulcanaydogan/AI-Provenance-Tracker) - Detects AI-generated content across text, images, audio, and video.

[**AI-Regulation-Compliance-Scanner**](https://github.com/ogulcanaydogan/AI-Regulation-Compliance-Scanner) - Checks AI systems against EU AI Act, UK AI Framework, and NIST AI RMF.
